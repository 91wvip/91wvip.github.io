# [超市的肉桂卷](https://91wvip.github.io/)

每周五晚八点，苏梅准时出现在超市烘焙区。

肉桂卷刚出炉，焦糖色的螺旋纹渗出琥珀糖浆。她取一只放托盘上，手腕微微发颤——不是老，是某种仪式性的郑重。

这个习惯始于女儿出国那年。机场送别后，她推着购物车在超市漫游，像失去锚的船。经过烘焙区时，那股温暖香气钩住了她：肉桂、黄油、烤化的红糖，混合成记忆里“家”的确切气味。

女儿从小爱吃肉桂卷。小学时，每个周五放学，小女孩会扑进厨房，鼻尖沾着粉笔灰：“妈妈，香死了！”面团在烤箱里膨胀，女儿趴在玻璃门上，呵出的白雾模糊了等待。那是苏梅最接近幸福的时刻——确切地说，是意识到自己正身处幸福之中的时刻。

女儿离家后，厨房成了纪念馆。烤箱冷得像墓碑。直到某个周五，超市的暖黄灯光收留了她。

她总是慢慢吃。先撕下一小块酥皮，在舌尖等它融化。肉桂粉微呛，像女儿青春期时的顶嘴。接着是面包体，柔软，带点韧劲——像那些母女拉锯的夜晚。最后是凝固的糖霜，甜得发苦，恰如思念本身。

今晚有点不同。旁边多了个年轻人。

女孩二十出头，染着蓝发，盯着肉桂卷发呆。苏梅看见她手腕的淤青，新旧叠加。“第一次买？”苏梅听见自己问。

女孩吓一跳，随即点头：“听说……这个能让人开心。”

苏梅多拿一个托盘：“要刚出炉的。凉了就不对了。”

她们在就餐区坐下。女孩狼吞虎咽，糖浆沾在嘴角。吃着吃着，眼泪掉进糖霜里。“对不起，”女孩抽泣，“我只是……想我妈了。”

苏梅递过纸巾。她想起女儿越洋电话里的鼻音：“妈，我昨晚梦到肉桂卷了。”一万公里外，女儿也在某个超市寻找同样的气味。

“我女儿和你差不多大。”苏梅说，“在伦敦。那儿也有肉桂卷，但她说味道不对。”

“为什么？”

“因为超市的肉桂卷，”苏梅轻拍胸口，“这里不对。”

女孩似懂非懂。但当她吃第二口时，动作慢了下来，开始品尝，而非吞咽。

苏梅讲起女儿：她如何第一次揉面团，如何把肉桂粉撒得到处都是，如何在烤焦时哭鼻子。往事在蒸汽中苏醒，她忽然看清——那些她曾以为琐碎的、重复的日常，原来是珍珠，被时间的线串成项链。

“你是个好妈妈。”女孩说。

苏梅摇头：“我只是个……一直在学习的妈妈。”

分别时，女孩问：“下周您还来吗？”

“来。”苏梅说，“每周五，雷打不动。”

女孩笑了：“那我教您个吃法——配咸奶油，我妈妈的做法。”

苏梅愣住。原来每个母亲都有自己的秘方，每个女儿都带着它走向远方。

回家路上，夜风裹着桂花香。她忽然想起女儿出生那天，产房窗外也有桂花。当时她想：我要把全世界的美好都给这个小人儿。

三十年后才明白，美好不是给予，是共享。像肉桂卷要撕开分食，甜才会在齿间弥漫。

手机震动，女儿发来照片：伦敦超市货架，一排肉桂卷。“妈，这个像你做的吗？”

苏梅放大图片，笑了。像不像不重要。重要的是，当女儿站在异国货架前，跨越时区想起了她。

就像此刻，她在这里，想起所有母亲和所有女儿——我们如何通过食物传递无法言说的爱，如何在陌生的城市寻找熟悉的气味，如何把思念烘焙成具体的形状，一口一口，吃成继续前行的力气。

烤箱提示音响起般，她心里“叮”了一声。
